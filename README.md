# Kwazam Chess ♟️

A custom-built strategy board game inspired by chess, developed in Java as part of an Object-Oriented Analysis & Design project. The game introduces unique piece types and movement rules, emphasizing clean architecture and scalable design.

---

## 🚀 Features

- Custom board game with original piece mechanics  
- Turn-based gameplay system  
- Distinct piece behaviors (Ram, Biz, Tor, Xor, Sau)  
- Visual representation using image assets  
- Move tracking via history system  
- Object-Oriented design (encapsulation, modularity)

---

## 🧠 Tech Stack

- **Language:** Java  
- **Paradigm:** Object-Oriented Programming (OOP)  
- **Assets:** Custom PNG images for game pieces  

---

## 🏗️ Project Structure
KwazamChess.java        # Main game logic and execution
